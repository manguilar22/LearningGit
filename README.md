# Learning Git
Learning Git , Journal

#### Starting Projects with Git 
```bash
git init <project_name> 
git init --bare<project_name> 
```

#### Using Git Config
```bash 
git config --global user.name "Kenny...x"
git config --global user.email "kenny1@email.com"
git config --list
```

#### Using Git Status
```bash
git status 
git status -s
git status -v 
```

#### Adding Files to a Project 
```bash 
git add . 
```
